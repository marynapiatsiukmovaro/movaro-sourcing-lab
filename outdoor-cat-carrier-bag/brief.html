Outdoor Cat Carrier / Lounge Bag · Анализ (выжимка) — вход в сорсинг

Бриф: выжимка анализа (Stage 1) для сорсинга

С чего стартуем в сорсинге (Stage 2) по этому продукту. Краткая выжимка из анализа — не сам анализ. Полный отчёт — в dossier (products/outdoor-cat-carrier-bag/intelligence/).

Что за товар

Раскладная переноска-трансформер (подтверждено видео). Закрытая = структурированная сумка-дюфель (мягкая ручка + плечевой ремень + двусторонние металлические молнии) — безопасно донести кота. Поставил и раскрыл — обе боковые панели раскладываются на землю в большой открытый сетчатый лежак + верхнее окно; кот ложится и смотрит на мир сквозь сетку. Внутри — съёмный карабин-страховка (анти-побег), усиленные швы («крепкая, не рвётся»), сетка по кругу. «Вмещает ~3 котов / есть где лечь». Серый, бирюзовый/оранжевый кант; размеры S / L. Ткань/каркас → COGS умеренный (AE ~$31; на Alibaba/1688 вероятно ~$20–25).

⚠️ Stage 1 (анализ) и Stage 2 (сорсинг) — раздельные этапы (не смешиваем; полный анализ — в dossier, intelligence-report). Внутри сорсинга свои шаги называем «Этап 0–3», чтобы не путать со Stage 1/2.
🔑 ВАЖНО (Marina): это НЕ точная копия конкурента. Конкуренты в той же нише, но предлагают другое/смежное (Travel Cat = рюкзаки-«снаряжение»; Kitastic = мягкая сумка/страх-ветеринар; Pawveras = bubble-рюкзак). Ближайший по концепту — Tuft&Paw Porto «переноска→лежак», но это всё равно НЕ наш раскладной трансформер. Товар выбран как свежая находка из категории AliExpress. → чистого «H1 точный клон» как у Karate Kitty может и не быть; планка НЕ ХУЖЕ держится на премиум-конструкции (Porto / ibiyaya), а не на копировании.

3 работы в одном (в этом моат — почему товар сильный)

1Открытый лежак на природе — поставил в парке/на пикнике, кот лежит и смотрит на мир рядом с тобой (его реальное поведение — наблюдать, не «гулять»).
2Спокойная переноска в машину/такси/ветеринар (инсайт Marina) — кот едет расслабленно, видит вокруг, есть место двигаться → не паникует и не гадит от тесной клетки.
3Домашнее «гнездо» — безопасное место дома.

Закрывает два главных непокрытых страха категории сразу: побег (закрытый режим + карабин) и жара/теснота (раскладной + сетка по кругу + верхнее окно).

«Не хуже» — чек-лист конструкции (планка качества)

① Анти-побег: запираемые двусторонние молнии + внутренний карабин-страховка (провал №1 категории — кот сбегает при загрузке).
② Раскладной open-air: откидные боковые панели + сетка по кругу + верхнее окно (анти-теснота/анти-жара — то, что закрытая переноска не может).
③ Жёсткое усиленное дно (не провисает/не качается) + мягкая ручка + плечевой ремень (комфорт хозяина).
④ Ткань без химозного запаха (low-VOC) — почти все воняют из коробки → дешёвый, решающий отличитель, если отгрузим без запаха.
⑤ Честный просторный размер (сесть И лечь) + реальная таблица размер/вес.
⑥ Nice-to-have: скатывающаяся шторка-навес (солнце + укрытие для пугливого кота).

Fix-list (из 1–3★ Amazon + форум-VOC — что чинить)

① Побег при загрузке (#1 провал — кот выскакивает до застёгивания / лезет в проёмы).
② Химозный/«рыбный» запах из коробки (почти каждый бренд → «проветривать днями»).
③ Жара + клаустрофобия — закрытые рюкзаки душные/тесные, кот в панике.
④ Нечестный размер (реальные габариты не совпадают) — недоверие по всей категории.
⑤ Комфорт хозяина + хлипкое дно (жёсткая доска в спину; провисший низ качается с котом).
⑥ Дропшип-ощущение (recycled-отзывы, junk-каталоги) → читается дёшево. Наш вес = премиум-FEEL.

Экономика (вход для B3)

Реклама ≈ $25+ (план на высокий CAC, ниша) · COGS вероятно ~$20–25 Alibaba/1688 (AE ~$31) · цель оффера: сингл ~$69–79 / value-бандл ~$89–99 (сумка + ДРУГой аксессуар + гайд «First Time Outside» + премиум-упаковка). US-ритейл $40–274. Цена = ВЫХОД из реального COGS, не наоборот.

⚠️ ВЕС → ДОСТАВКА = критично для маржи (Marina). Структурированная раскладная сумка тяжелее тканевой игрушки → выше landed/доставка → бьёт в маржу. Обязательно достать реальный вес + реальную доставку и заложить в landed COGS ДО фиксации цены. Это же поддерживает верх диапазона ($79/$99), не дно.

2 новые задачи сорсинга (Marina — сверх обычного)

1Бандл-аксессуар = ДРУГой товар (не 2 одинаковых): шторка-навес от солнца / коврик-подстилка под пару / клипса-страховка-поводок. Желательно из каталога того же финального поставщика (одна посылка). Вовлечение + value-stack ($89–99).
2Премиум-упаковка — спросить у поставщика/агента нормальную ритейл-упаковку. Категория шлёт дёшево → упаковка = реальный отличитель + наш премиум-FEEL против дропшип-полосы (планка = магазин Pro Cat Wrap).

Стратегия сорсинга (согласована)

Берём проверенное (что работает у премиум-конкурентов) — товар НЕ ХУЖЕ Porto / ibiyaya (равный-хороший / другой / лучше; «улучшенный» желателен, не обязателен). Качество: только ХОРОШИЙ товар (с плохого нет адекватного фидбэка рынка). Дешёвый дубль = метим на карте, НЕ запускаем.

Гипотезы H1–H6 (едины с хабом сорсинга)

H1точный клон класса — раскладной конвертибл: откидные боковины + сетка + верхнее окно + карабин (наш товар!)
H2та же идея, лучше конструкция — усиленное дно, лучше молнии/карабин, low-VOC ткань, прочнее сетка
H3кандидаты в бандл-аксессуар (шторка-навес / коврик-подстилка / клипса-поводок — ДРУГой товар, тот же поставщик)
H4дешёвые дубли — помеченное «дно», низкий приоритет (не запускаем, просто видим на карте)
H5неожиданные крутые находки — лучшие конвертибл/раскладные дизайны, премиум-материалы, умные фичи
H6смежные формы — раскладной рюкзак, переноска под коляску, dog/dual-версия (потенциал расширения)
📐 Метод: фото + «делает ли нашу задачу (закрыл-донёс / раскрыл-лежак, анти-побег, открытый воздух)» = первично, имя = подсказка. Один товар = одна позиция (тот же товар у разных продавцов — группируем). Цены из листинга врут → клик-сверка только у финалистов. Полный алгоритм и уроки — в хабе сорсинга.
