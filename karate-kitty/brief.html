Karate Kitty · Анализ (выжимка) — вход в сорсинг

Бриф: выжимка анализа (Stage 1) для сорсинга

С чего стартуем в сорсинге (Stage 2) по этому продукту. Краткая выжимка из анализа — не сам анализ.

Что за товар

Wearable arm-puppet — мягкая «перчатка-кукла» в виде кота, которую владелец надевает на руку; кот борется, кусает и бьёт лапами её, а не голую кожу хозяина. Внутри — мягкий наполнитель, кошачья морда, «хрустящие» уши, кармашки с кошачьей мятой, защитный рукав на предплечье. Ткань → низкий COGS (дубли на Amazon $14–23 → landed ~$5–10).

⚠️ Stage 1 (анализ) и Stage 2 (сорсинг) — раздельные этапы (не смешиваем; полный анализ — в dossier продукта, intelligence-report). Внутри сорсинга свои шаги называем «Этап 0–3», чтобы не путать со Stage 1/2.

«Не хуже» — чек-лист конструкции (планка качества)

① Прочный защитный рукав (когти не проходят — у CATSUMO 600D oxford nylon)
② Плюш + кошачья морда + crinkle (хрустящие уши) — вовлечение
③ Кошачья мята (в идеале сильная / пополняемая)
④ БЕЗ мелких отрывных деталей (их дефект №1 + риск подавиться)
⑤ Эргономичные каналы под пальцы (управление лапами)
⑥ Масштаб под КОТА, не под собаку (их дубли «dog-sized» — реальная жалоба)

Fix-list (из 1–3★ отзывов — что чинить)

① Размер/масштаб (#1 негатив — «слишком большой, пугает кота»; ни у кого нет size-guide)
② Вовлечение не универсально (~30–40% «кот проигнорировал / испугался»); мята выдыхается, не пополняется
③ Отрывные детали (колокольчики/усы/нос отваливаются за дни = choke)
④ Эргономика (узкие/отсутствующие отверстия под пальцы → плохо играть)
⑤ Честность (KK: 1237 отзывов сверху vs 553 в виджете; «vet approved» без имени врача)

Экономика (вход для B3)

Реклама ≈ $25 фикс (план на высокий CAC) · COGS перчатки низкий (~$5–10 ткань) · цель оффера: сингл ~$45–49 / value-бандл ~$59–64 (перчатка + 2-я ДРУГАЯ игрушка + гайд + премиум-упаковка). Категория $37–50, дно дублей $14–23. Цена = ВЫХОД из реального COGS, не наоборот.

💡 Слабость конкурента (Marina): Karate Kitty гонит BOGO — 1+1 ОДИНАКОВЫХ игрушки за ~$60 AOV. Зачем покупателю две одинаковые? Наш бандл = 2 РАЗНЫЕ игрушки за те же ~$59 → выше воспринимаемая ценность при том же чеке. Это и есть зачем сорсим 2-ю ДРУГУЮ игрушку.

2 новые задачи сорсинга (Marina — сверх обычного)

12-я ДРУГАЯ физическая игрушка для бандла (другой стиль игры: kicker / дразнилка / электроника). Желательно из каталога того же финального поставщика (одна посылка/логистика). Хедж вовлечения + value-stack.
2Премиум-упаковка «как у iPhone» — спросить у поставщика/агента нормальную ритейл-упаковку. Категория шлёт дёшево → упаковка = реальный отличитель (планка = магазин Pro Cat Wrap).

Стратегия сорсинга (согласована)

Берём проверенное (что работает у конкурента) — товар НЕ ХУЖЕ Karate Kitty / CATSUMO (равный-хороший / другой / лучше; «улучшенный» желателен, не обязателен). Качество: только ХОРОШИЙ товар (с плохого нет адекватного фидбэка рынка). Дешёвый дубль = метим на карте, НЕ запускаем.

Гипотезы H1–H6 (едины с хабом сорсинга)

H1точный клон класса — arm-puppet + защитный рукав + кошачья морда + мята (класс Karate Kitty / CATSUMO)
H2та же идея, другое качество/конструкция (600D oxford vs тонкий рукав; усиленный vs дешёвый)
H3кандидаты во 2-ю игрушку (другой стиль игры: catnip-kicker, дразнилка/wand, электроника)
H4дешёвые дубли — помеченное «дно», низкий приоритет (не запускаем, просто видим на карте)
H5неожиданные крутые находки — интересные interactive/enrichment игрушки (потенциал: 2-я игрушка / будущий товар). Сигнал = wow, не ширпотреб
H6смежные формы + кросс-вид (собака) — собачья версия / dog+cat дуал / другие конструкции той же идеи
📐 Метод: фото + «делает ли нашу задачу (безопасная контактная игра)» = первично, имя = подсказка. Один товар = одна позиция (тот же товар у разных продавцов — группируем). Цены из листинга врут → клик-сверка только у финалистов. Полный алгоритм и уроки — в хабе сорсинга.
