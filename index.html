Поиск и валидация реального товара — по стейджам. Продукт: Pro Cat Wrap (самоклеящийся успокаивающий wrap для кошек).

⚠️ Промежуточная база, не финал. Здесь живёт «полевая» работа Stage 2 со ссылками; финальное решение переезжает в чистый дossier продукта. Обновляется по ходу.

Маршрут Stage 2 (стратегия)

  1. Якорь: берём точный товар сильного конкурента (CalmCozyCat) — под него ищем «точь-в-точь».
  2. Рынок + варианты: сканируем категорию и все конструкции (самоклейка / ремешки-velcro / замок / mesh-мешок), не отсекаем заранее.
  3. Шорт-лист источников: реальные поставщики (Alibaba / AliExpress / 1688) — карточки с ценой/MOQ/ссылкой.
  4. Углубление и сверка с конкурентом → основной + резервный + экономика + размерный ряд (= финал).
