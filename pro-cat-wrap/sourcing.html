Сорсинг — всё в одном месте

Цель: товар равный CalmCozyCat или лучше для теста. Метод (на каждой площадке): фото-поиск + ключевые запросы → классификация H1–H6 → карта-отбор галочками → Stage-2 разбор. Финал — на объединённом пуле всех площадок.

🚧 СЫРАЯ БАЗА. Картинка + «делает ли нашу задачу (фиксация для стрижки)» = первично; имя = подсказка; цены из листинга ненадёжны → сверяем кликом только у финалистов. Площадки НЕ финализируем по одной — собираем все, курируем в конце.
