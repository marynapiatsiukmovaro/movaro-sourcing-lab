Pro Cat Wrap · Анализ (выжимка) — вход в сорсинг

Бриф: выжимка анализа (Stage 1) для сорсинга

С чего стартуем в сорсинге (Stage 2) по этому продукту. Это краткая выжимка из анализа — не сам анализ.

Зачем мы здесь

Stage 1 (анализ) по продукту завершён, продукт признан интересным для теста. Stage 2 (сорсинг) = найти конкретный физический товар, которым реально торгуем: поставщик, экономика, характеристики, размеры, комплектация.

⚠️ Stage 1 (анализ) и Stage 2 (сорсинг) — раздельные этапы (мы их не смешиваем; полный анализ живёт отдельно — в dossier продукта, intelligence-report). Внутри сорсинга свои шаги называем «Этап 0–3», чтобы не путать со Stage 1/2.

Выжимка из Stage 1 (что уже знаем)

Конкурент-якорьCalmCozyCat — победитель категории; под его товар ищем «точь-в-точь».
Реальный товар конкурента (уточнено в сорсинге)флис/фетр + молния + велкро-панели + кармашки для лап (премиум $79–119). Это НЕ дешёвый плоский self-adherent свадл.
Позиционирование«Calm, not trapped» — wrap успокаивает кота для стрижки когтей / лекарств / груминга дома.
Fix-list (из отзывов)① переклеивается/трудно раскрыть · ② мал по размеру / кот вырывается · ③ ворс-линт · ④ «нужен уже спокойный кот».

Полный отчёт Stage 1 (intelligence-report + RU-HTML) — в dossier продукта. B5 (финансы) и B6 (наука) проверены ✓.

Задачи сорсинга (Stage 2)

B4реальные спеки + размерный ряд (материал, застёжка, размеры по весу)
B3поставщики (3–5) + экономика (COGS → жизнеспособный AOV)
B2подарки (цифровые: ценный гайд + памятка)
Итогосновной + резервный кандидат → переезжает в чистый dossier

Стратегия сорсинга (согласована)

Берём проверенное (что работает у конкурента) — товар равный CalmCozyCat или лучше; при дешёвой возможности улучшаем → тест. Конвертит за ~2 дня → углубляемся за лучшими вариантами. Идеал в открытом доступе = поздно; сила — взять рабочее, протестировать, потом углублять. Качество: только ХОРОШИЙ товар (с плохого нет адекватного фидбэка).

Гипотезы H1–H6 (едины с хабом сорсинга)

H1точный клон — флис + молния + велкро + кармашки (класс CalmCozyCat; якорь Vetnutro)
H2self-adherent обёртки/фиксаторы для стрижки (массовый класс)
H3anxiety-жилет / apron (ThunderShirt-класс)
H4mesh / bath-мешок (кот в мешок, голова/лапы наружу)
H5recovery / доп-фичи (после операции, 3-в-1)
H6прочие формы (гамак, плёнка, towel-burrito)
📐 Метод: фото + «делает ли нашу задачу» = первично, имя = подсказка. Один товар = одна позиция (тот же товар у разных продавцов — группируем). Полный алгоритм и уроки — в хабе сорсинга.
