Pro Cat Wrap · Stage 0

Бриф сессии + отчёт Stage 1

Что готовим и с чего стартуем во втором этапе по этому продукту.

Зачем мы здесь

Этап 1 (анализ) по продукту завершён и продукт признан интересным для теста. Этап 2 = найти конкретный физический товар, которым мы реально торгуем: поставщик, экономика, характеристики, размеры, комплектация.

⚠️ Этап 1 (анализ) и этап 2 (сорсинг) — раздельные этапы. Здесь мы их не смешиваем; полный анализ живёт отдельно (dossier продукта, intelligence-report).

Выжимка из Stage 1 (что уже знаем)

Конкурент-якорьCalmCozyCat — победитель категории; под его товар ищем «точь-в-точь».
Позиционирование«Calm, not trapped» — wrap успокаивает кота для стрижки когтей / лекарств / груминга дома.
Fix-list (из отзывов)① переклеивается/трудно раскрыть · ② мал по размеру / кот вырывается · ③ ворс-линт · ④ «нужен уже спокойный кот».

Полный отчёт Stage 1 (intelligence-report + RU-HTML) — в dossier продукта. B5 (финансы) и B6 (наука) проверены ✓.

Задачи этой сессии (Stage 2)

B4реальные спеки + размерный ряд (материал, застёжка, размеры по весу)
B3поставщики (3–5) + экономика (COGS → жизнеспособный AOV)
B2подарки (цифровые: ценный гайд + памятка)
Итогосновной + резервный кандидат → переезжает в чистый dossier

Стратегия сорсинга (согласована)

Берём проверенное (что работает у конкурента), при дешёвой возможности улучшаем → тест. Конвертит за ~2 дня → углубляемся за лучшими вариантами. Идеал в открытом доступе = поздно; сила — взять рабочее, протестировать, потом углублять.

Критерии «товара-победителя» (scorecard)

размерный ряд по весу · лёгкость открытия (жалоба №1) · ткань/линт · анти-escape · цена/MOQ/OEM · отзывы/качество · потенциал брендинга.

Гипотезы поиска (находим и сравниваем — не один вариант)

H1точный товар конкурента — self-adherent свадл (baseline)
H2улучшенный self-adherent (лучше ткань/низкий линт, размеры, OEM)
H3velcro-жилет на ремешках (ThunderShirt-класс)
H4на замке / zip
H5wrap с доп-фичами (лимб-флапы, шея, 3-в-1, kit)
H6mesh-мешок / гамак (иная механика, фон)
📐 Правило шорт-листа: по каждой гипотезе показываем 5–10 лучших ссылок ДО финального отбора. Один товар = одна позиция (если тот же товар у разных продавцов — группируем рядом, не выдаём за разные находки).
